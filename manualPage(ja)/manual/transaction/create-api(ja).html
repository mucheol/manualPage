APIキーはどのように発給されますか？

1. メイン画面の右側上段の人アイコン洗濯。
2. API管理をクリック。
3. 様式を記入。
  • IP addressを正確に記入してください。
  • verification codeとgoogle authentication code認証が終わると、
    create APIが完了されます。
4. API record
  • create apiが完了すると、画面下段のapirecordに情報が現れます。
