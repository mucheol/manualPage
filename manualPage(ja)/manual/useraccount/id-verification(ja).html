実名認証はどうしますか。

1. メイン画面の右側上段の人アイコンクリック。
2. ID Verification欄からVerificationクリック。
3. 国または地域を選択します。
4. Certified ID Type 選択
5. 様式作成
  • ID NO.はアップロードする写真と比較, 確認するため、正確に作成してください。
6. 写真 アップロード
  • 右の画像のように写真を撮ってアップロードしてください。
  • 該当画面の下の注意事項をすべてお読みください。
  • アップロードが完了されたらSubmitボタンを押して完了してください。
7. 審査待ち
  • 前の段階で入力した証明番号と身分証明書、画像を照らし合わせて審査に入ります。
